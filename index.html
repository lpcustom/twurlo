Welcome to the Twurlo Github Page. Twurlo is an open source link redirection script in PHP. It can be used on most PHP enabled hosts with little effort. The main goal behind Twurlo is to allow site owners to utilize their site as a proxy for links. This allows them to spread brand recognition of their site and to track statics on link click-throughs. In future releases we will be adding the ability to show a quick advertisement to the users before sending them to the destination link.
